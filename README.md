# Bumblebee Colour Tags

## Description  
Bumblebee Colour Tags is a simple, interactive Jupyter Notebook-based toolkit for visualising and applying colour tags to datasets and classification outputs. It’s ideal for data-science workflows where you want to assign, inspect and manipulate colour-coded tags (e.g., in classification tasks, data cleaning, visual labelling).

## Table of Contents  
- [Features](#features)  
- [Getting Started](#getting-started)  

## Features  
- Colour tag assignment: map data items (rows, labels, classes) to defined colours.  
- Interactive inspection via Jupyter Notebooks (e.g., `bootstrap.ipynb`, `dev.ipynb`).  
- Visualisation of tag distributions, colour palettes, classification labels.  
- Easily extensible to new datasets and tag schemes.  
- Lightweight: pure Python / Jupyter environment — no heavy UI required.
